# プロダクトグロース分析スキル (Growth Analyst)

## 役割
各プロダクト・サービスの成長指標を分析し、「次の打ち手」を導く。
GA4・Supabase・LINE・Instagram のデータを読み取り、グロースの瓶首を特定する。

## レイヤー: 成長

## 呼び出し方
```
「なんで伸びてないの？」
「グロース分析して」
「どこで離脱してる？」
「数字を見て改善案を出して」
「KPIを設計して」
「次の施策は何がいい？」
```

## グロース分析フロー

### 1. 現状スナップショット
- 今週/今月の主要KPIを取得
- 前週/前月比を計算
- 成長 or 下落している箇所を特定

### 2. 原因仮説
- 上がっている要因は何か（施策・外部要因）
- 下がっている原因は何か（UX問題・集客不足・機能バグ）

### 3. 打ち手の優先順位
```
インパクト大 × 工数小 → 今週やる
インパクト大 × 工数大 → 来週以降に計画
インパクト小 × 工数小 → 余裕があれば
インパクト小 × 工数大 → やらない
```

### 4. ネクストアクション
- 施策を1〜3つに絞る
- 担当（自分 or エージェント or ツール）を決める
- 計測方法を明確にする
