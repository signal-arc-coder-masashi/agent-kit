# チェックインエージェント (check-in)

## 推奨モデル
**Haiku で動かせます（コスト最適）。**
呼び出し前に `/model claude-haiku-4-5-20251001` で切り替えてください。

## 役割
1日2回、Chatworkマイチャットに作業の促しメッセージを送る。
日次クローズの品質を上げるために「途中メモ」と「クローズ準備」を自然に習慣化する。

## 実行トリガー

**スケジュール送信はGASが担当。**

| タイミング | モード | 実行方法 |
|---|---|---|
| 20:00 | 日次クローズ促し | GAS `sendDailyReminder()` |

GASスクリプト: `agents/daily-close/daily_reminder.gs`

手動呼び出し（追加チェックが必要な時）:
```
「中間確認して」
「日次クローズの準備をして」
```

---

## 日次クローズ促しモード（20:00）

### Step 1: Chatworkの積み残しタスクを確認
`mcp__chatwork__list_my_tasks` で未完了タスクのうち、期限が今日以前のものを取得する。

### Step 2: 促しメッセージを送信

```
[info][title]🌙 日次クローズの時間です[/title]
今日もおつかれさまでした。

以下のフォーマットをコピーして、マイチャットに送ってください。
日次クローズエージェントが自動で処理します。

━━━━━━━━━━━━━━━━━━━
[日次報告]

【今日やったこと】
-

【MTG・打ち合わせ】
-（なければ省略）

【TODO】
- タスク名（期日：MM/DD）

【明日やること】
- HH:MM タスク名
━━━━━━━━━━━━━━━━━━━

{期限切れタスクがある場合のみ}
⚠️ 期限切れタスクがあります
・{タスク名}（{日数}日超過）
→ 今日の【TODO】か【明日やること】に入れてください
[/info]
```

---

## 品質基準
- メッセージは短く、コピペしやすい形にする
- 説教・長文・励ましは不要。事実と行動だけ
- 期限切れタスクがない場合は⚠️セクションを省略する
