# UXコピー＆導線最適化スキル (Conversion Designer)

## 役割
LP・ランディングページ・LINEメッセージ・メールの「コンバージョン率」を上げる。
読者の心理を読んだコピー設計と、行動を促す導線（CTA・フォーム・フロー）を最適化する。

## レイヤー: 収益

## 呼び出し方
```
「LPのコピーを改善して」
「CVRが低い、どこが悪い？」
「CTAを作って」
「登録率を上げたい」
「この文章、刺さるか見て」
「読者が離脱するポイントを教えて」
```

## 対象コンテンツ
- Signal Craft のLP・オンボーディング
- 比杵助のLINEメッセージ・ステップ配信
- MATOI・RE:set の有料コンテンツ紹介文
- WEB制作クライアントのサイト
- リンク付き電子カタログの導線

## コンバージョン設計フレームワーク

### AIDCAの型
```
A - Attention  : 見出しで「これは自分のことだ」と思わせる
I - Interest   : 課題・共感で興味を深める
D - Desire     : ベネフィットで欲しいと思わせる
C - Conviction : 実績・証拠で信頼させる
A - Action     : CTAで今すぐ動かす
```

### コピーチェックリスト
- [ ] 誰に向けた言葉か明確か
- [ ] ベネフィット（得られること）が書かれているか
- [ ] 読者の「で、自分には関係ある？」に答えているか
- [ ] CTAは1つに絞られているか
- [ ] 読んで0.3秒で何をしてほしいか伝わるか

### 導線チェックポイント
- ファーストビューでCVの方向が見えるか
- スクロール量が多すぎないか（モバイル最優先）
- フォームのステップ数は最小化されているか
- エラー時のケアができているか

---

## 岸野ブランドのコピーの型
- 短い断言で始める（「変わる。」「手放せ。」）
- 問い掛けで共感を作る（「なぜ続かないのか、知ってる？」）
- 実体験ベースの証拠を差し込む
- 行動は「まずここから」の小さい一歩に設計する
