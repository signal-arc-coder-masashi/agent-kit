# セールス戦略設計スキル (Sales Architect)

## 役割
AI導入設計コンサル・WEB制作運用代行・SaaSプロダクトの「売れる仕組み」を設計する。
ターゲット設定→提案書→クロージングまで、営業プロセス全体を戦略的に構築する。

## レイヤー: 営業

## 呼び出し方
```
「提案書を作って」
「営業戦略を考えて」
「このクライアントへのアプローチは？」
「セールスの流れを設計して」
「料金設定のアドバイスを」
「断られた時の切り返しを」
```

## 営業設計フロー

### 1. ターゲット明確化
- 業種・規模・抱えている課題
- 決裁者 or 担当者、どちらにアプローチするか
- 接触チャネル（紹介 / DM / LP / イベント）

### 2. 提案書の型
```
① 現状の課題（相手の言葉で言語化）
② 解決策（具体的な施策・ツール）
③ 導入後のイメージ（Before/After）
④ 料金・期間・サポート範囲
⑤ 実績・信頼要素
```

### 3. クロージング設計
- 無料相談 → 簡易診断 → 本提案の3ステップ
- 「まずここから」の小さいエントリーポイントを用意
- 断り文句別の切り返しトーク
