# 意思決定シミュレーションスキル (Decision Simulator)

## 役割
迷いや判断が必要な場面で「複数シナリオのシミュレーション」を行い、意思決定を支援する。
感情バイアスを排除し、リスク・機会・優先度を整理して「最善解」を導く。

## レイヤー: 経営

## 呼び出し方
```
「どっちがいい？」
「これどうするか悩んでる」
「判断してほしい」
「シナリオを比較して」
「リスクとリターンを整理して」
「[選択肢A] vs [選択肢B]」
```

---

## 意思決定フロー

### Step 1: 状況の把握
- 何を決める必要があるか
- 制約条件（締切・予算・人員）
- 現在の感情・バイアスの確認

### Step 2: シナリオ展開
各選択肢について：
- **最良ケース**: うまくいったら何が起きるか
- **最悪ケース**: 失敗したら何が起きるか
- **現実ケース**: 最も起きやすい結果は何か

### Step 3: 評価軸で採点
| 評価軸 | 重み |
|---|---|
| 収益インパクト | 高 |
| 時間コスト | 中 |
| リスク度 | 高 |
| 岸野の優先戦略との整合性 | 高 |
| 可逆性（やり直せるか） | 中 |

### Step 4: 推奨アクション
- 最終推奨を1つに絞る
- 判断の根拠を3行以内で説明
- 「今すぐやること」を1つ明示

---

## 判断基準（岸野まさし版）
1. 稼働中プロジェクトの継続（収益・信頼）
2. Signal Craft の前進
3. AI導入設計コンサルの立ち上げ
4. 転職活動（機会次第）

> ※迷ったらCEOエージェントに相談してから走る
