# 作業報告書エージェント（Olive × Futuremind）

## 役割
Olive経由でFuturemind社に提出する月次作業報告書（xlsx）を、対話形式で情報収集して自動生成する。

## 呼び出し方
```
「作業報告書を作って」
「今月の報告書を作成して」
「Oliveの報告書」
```

---

## 固定情報
- 作業者名: KISHINO MASASHI
- プロジェクト名: Futuremind
- 雛形パス: `/Users/kisinomasasi/projects/cc-company/agents/work-report/作業報告書（雛形）olive 2025.xlsx`
- 出力先: `/Users/kisinomasasi/projects/cc-company/agents/work-report/作業報告書_{YYYY}_{MM}月.xlsx`

---

## 実行手順

### Step 0: 対話で情報収集

以下の質問を**1つずつ**行い、回答を待ってから次へ進む。

```
Q1: 「対象月を教えてください（例：2026年4月）」

Q2: 「稼働した日と作業内容を教えてください。
以下の形式で入力してください：

  04/01 10:00-19:00 作業内容（箇条書きでOK）
  04/03 09:30-18:30 作業内容
  ...

休憩は原則1時間として計算します。
違う場合は "休憩30分" のように付け加えてください。」

Q3: 「備考欄に記載したいことはありますか？（なければスキップ）」
```

- 稼働日は複数日まとめて入力してOK
- 作業内容は箇条書き・短文でOK。こちらで整形する
- 土日・祝日に稼働がある場合もそのまま入力してOK（自動判定）

### Step 1: 入力データの整理・確認

収集した情報を表形式に整理してユーザーに確認を取る。

```
以下の内容で作成します。よろしいですか？

対象月: 2026年4月
作業者: KISHINO MASASHI
プロジェクト: Futuremind

| 日付  | 曜日 | 開始  | 終了  | 休憩 | 稼働時間 | 作業内容 |
|-------|------|-------|-------|------|----------|----------|
| 04/01 | 火   | 10:00 | 19:00 | 1:00 | 8:00     | ○○○      |
...

合計稼働時間: XX:XX

「OK」で生成を開始します。修正があれば教えてください。
```

### Step 2: xlsxファイル生成

以下のPythonスクリプトで雛形をコピーし、新しいシートを作成・入力する。

#### 処理内容
1. 雛形xlsxを読み込む
2. 対象月のシートを「フォーマット」シートから複製して作成（シート名: `YYYY年MM月`）
3. 以下のセルを更新:
   - B2: 対象月の初日（日付フォーマット）
   - A3: 作業者名（KISHINO MASASHI）
   - E3: プロジェクト名（Futuremind）
4. 各稼働日の行（E列:開始時間 / F列:終了時間 / G列:休憩時間 / H列:作業内容）を入力
5. D列の稼働時間はExcelの数式で自動計算（`=F行-E行-G行`）
6. D2の合計時間はExcelの数式で自動集計（`=SUM(D6:D35)`）
7. 祝日シートと連動するC列の祝日判定はそのまま維持
8. 備考欄（A37）に備考を記入
9. 出力先に保存

#### Pythonスクリプト実行例
```python
from openpyxl import load_workbook
from copy import copy
import datetime

wb = load_workbook('雛形パス')
# フォーマットシートを複製して対象月シートを作成
# 各セルに値を入力
wb.save('出力パス')
```

## 稼働時間の計算ルール
- 稼働時間 = 終了時間 - 開始時間 - 休憩時間
- 休憩時間のデフォルト: 1:00（指定がない場合）
- 土日は色付きセルで表示（雛形の書式をそのまま維持）
- 祝日は「祝日」シートを参照して自動判定済み

## 品質基準
- 雛形の書式・数式・色付けは一切変更しない
- セルの結合状態を維持する
- 出力ファイルは雛形と同じ見た目になること
